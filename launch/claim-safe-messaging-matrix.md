# Claim-safe messaging matrix

Use this matrix before copying launch text into Product Hunt, GitHub, X, LinkedIn or future web CTA surfaces.

| Allowed claim | Safer wording | Forbidden stronger version | Why |
|---|---|---|---|
| Local/BYOK path is documented. | The local/BYOK path is planned and documented. | Forbidden: run BYOK capsules today. | Execution is not implemented in this repo. |
| Public manifests preview future templates. | Public manifests preview future template directions. | Forbidden: templates are executable. | C03 added manifest-only previews, not runnable templates. |
| Config scaffold exists. | C04 adds placeholder-only config scaffolding. | Forbidden: provider is configured. | No provider setup or execution exists. |
| Install promise is visible. | The current action is clone and validate. | Forbidden: install the launcher with a one-click setup. | No installer, binary or executable launcher exists. |
| Cloud handoff is future path. | TransformIA Cloud is a future handoff path. | Forbidden: Cloud sync is active. | No Cloud sync, tenant state or customer portal exists. |
| Source-available license. | The repo uses a source-available evaluation license. | Forbidden: open source. | Public visibility does not grant broad reuse rights. |
| Private core boundary. | Private runtime and Exocortex core are not included. | Forbidden: private runtime included. | The launcher repo must not expose private runtime code. |
| Validation exists. | Validators check public repo boundaries and false claims. | Forbidden: validators prove production readiness. | Validators protect docs posture, not runtime production behavior. |
| Execution boundary. | No local capsule execution is included yet. | Forbidden: executable launcher is available. | C05 adds copy assets only. |
| Provider config boundary. | Provider placeholders remain non-executable. | Forbidden: provider is configured. | Real provider setup and keys are absent. |
| Payment boundary. | No payment or Stripe flow is included. | Forbidden: Stripe live. | Payment implementation is out of scope. |
| Production readiness boundary. | The repo is launch-copy ready for human review. | Forbidden: this launcher is production ready. | Production capability is not implemented or approved. |
