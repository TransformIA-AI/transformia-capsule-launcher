# Launch assets

This folder contains draft public launch assets for TransformIA Capsule Launcher.

These files are copy drafts only. They are not execution artifacts, submission scripts, campaign automation, analytics configuration or tracking infrastructure.

## Boundaries

- No paid campaign is included.
- No Product Hunt submission has been performed by this repository.
- No posting to X or LinkedIn has been performed by this repository.
- No analytics, tracking pixel, UTM campaign or URL shortener is included.
- No tracking is included.
- No external network calls are introduced by these assets.
- No private TransformIA runtime, Exocortex core, connector workflow, customer material, prompt file or secret is included.
- All launch claims must remain source-available/IP-safe and aligned with `LICENSE.md`, `NOTICE.md` and the public boundary docs.

## Use

Use these drafts as human-reviewed starting points for a future launch moment. Before copying any text into a public channel, re-run repository validators and review [`launch/claim-safe-messaging-matrix.md`](claim-safe-messaging-matrix.md).
